AXTIV 사용자 가이드
================================================================================
버전: 1.0
최종 업데이트: 2025-10-10

================================================================================
1. 시작하기
================================================================================

1.1 로그인 및 회원가입
  - 홈페이지(/) 접속
  - "AI 도구 사용하기" 또는 "내 프로젝트로 이동" 버튼 클릭
  - Google 계정으로 로그인

1.2 첫 회사 생성 또는 가입
  - 로그인 후 회사 검색 페이지로 자동 이동
  - 옵션 1: 기존 회사 검색 및 가입
    * 검색창에 회사명 입력
    * 검색 결과에서 원하는 회사 선택
    * "가입하기" 버튼 클릭
  - 옵션 2: 새 회사 생성
    * "새 회사 생성하기" 버튼 클릭
    * 회사명 입력 (필수)
    * 회사 설명 입력 (선택)
    * "회사 생성" 버튼 클릭

================================================================================
2. 회사 관리
================================================================================

2.1 회사 검색
  - /companies 페이지에서 검색 가능
  - 검색창에 회사명 입력 (자동 검색)
  - 검색 결과에서 회사 정보 확인:
    * 회사 로고
    * 회사명
    * 회사 설명
    * 생성일

2.2 회사 가입
  - 검색 결과에서 "가입하기" 버튼 클릭
  - 이미 다른 회사에 소속된 경우 변경 확인 메시지 표시
  - 가입 완료 후 자동으로 워크스페이스 페이지로 이동

2.3 회사 전환
  - 현재 소속된 회사를 변경하려면:
    * 다른 회사를 검색하여 "가입하기" 클릭
    * 확인 메시지에서 "확인" 선택

================================================================================
3. 워크스페이스 관리
================================================================================

3.1 워크스페이스 생성
  - 회사 페이지에서 "새로 만들기" 버튼 클릭
  - 워크스페이스명 입력 (필수)
  - 워크스페이스 설명 입력 (선택)
  - "워크스페이스 생성" 버튼 클릭
  - 생성 시 자동으로 "일반" 채팅방 생성됨

3.2 워크스페이스 탭 전환
  - "내 워크스페이스": 가입한 워크스페이스만 표시
  - "워크스페이스 검색": 모든 워크스페이스 검색 및 가입 가능

3.3 워크스페이스 가입
  - "워크스페이스 검색" 탭 선택
  - 검색창에 워크스페이스명 입력
  - "가입하기" 버튼 클릭

3.4 워크스페이스 정보 확인
  - 각 워크스페이스 카드에 표시되는 정보:
    * 워크스페이스명
    * 회사명
    * 멤버 수
    * 사용자 역할 (생성자/관리자/멤버)
    * 가입 상태 (가입됨/미가입)

================================================================================
4. 채팅 기능
================================================================================

4.1 채팅방 목록
  - 워크스페이스 입장 시 왼쪽에 채팅방 목록 표시
  - 각 채팅방 정보:
    * 채팅방 이름
    * 멤버 수
    * 마지막 활동 시간
    * 새 메시지 알림 (빨간 점)
    * "기본" 배지 (기본 채팅방인 경우)

4.2 채팅방 생성
  - 권한: 워크스페이스 생성자 또는 관리자만 가능
  - 채팅방 목록 하단 "채팅방 추가" 버튼 클릭
  - 채팅방명 입력
  - 채팅방 설명 입력 (선택)
  - "생성" 버튼 클릭

4.3 일반 채팅
  - 채팅방 선택
  - 하단 입력창에 메시지 입력
  - "전송" 버튼 클릭 또는 Enter 키 누르기
  - Shift + Enter: 줄바꿈

4.4 AI 채팅 모드
  - 하단 입력창 우측의 로봇 아이콘(🤖) 클릭
  - AI 모델 선택:
    * GPT-4o (기본값)
    * GPT-4o-mini
    * GPT-3.5-turbo
    * Claude-3.5-sonnet
    * Claude-3.5-haiku
  - 메시지 입력 후 전송
  - AI가 자동으로 응답 생성
  - Claude 모델 선택 시 실시간 스트리밍 답변 표시

4.5 파일 업로드
  - 입력창 우측 클립 아이콘(📎) 클릭
  - 파일 선택 또는 드래그 앤 드롭
  - 지원 파일 형식:
    * 이미지: PNG, JPG, GIF 등
    * 문서: PDF, DOC, DOCX, TXT, MD
    * 압축: ZIP, RAR
    * 동영상: MP4, MOV, AVI
  - 파일별 최대 크기: 10MB
  - 한 번에 최대 5개 파일 업로드 가능
  - AI 모드에서 이미지 및 PDF 파일 분석 가능

4.6 파일 메시지 기능
  - 이미지 파일:
    * 썸네일 자동 생성
    * 클릭하여 확대 보기
    * 다운로드 버튼 제공
  - 일반 파일:
    * 파일 이름, 크기 표시
    * 파일 타입별 아이콘 표시
    * 다운로드 버튼 제공

4.7 채팅방 참여 인원 확인
  - 채팅방 헤더의 참여 인원 버튼(👥 숫자) 클릭
  - 참여 인원 목록 다이얼로그 표시:
    * 사용자 아이콘
    * 사용자 이름
    * 이메일
    * 관리자 배지 (관리자인 경우)
    * 현재 사용자 강조 표시

4.8 채팅방 나가기
  - 채팅방 헤더 우측의 나가기 버튼 클릭
  - 확인 다이얼로그에서 "나가기" 선택
  - 기본 채팅방은 나갈 수 없음

4.9 메시지 표시 기능
  - 실시간 연결 상태 표시 (초록색 점)
  - 타임스탬프 자동 표시
  - 사용자별 프로필 아이콘
  - AI 메시지는 마크다운 형식으로 표시:
    * 제목 (H2, H3)
    * 목록 (순서 있음/없음)
    * 코드 블록
    * 인라인 코드
    * 링크 (새 탭에서 열림, 파란색으로 강조)

4.10 무한 스크롤
  - 채팅 목록 상단으로 스크롤 시 이전 메시지 자동 로드
  - 한 번에 50개씩 메시지 로드

5. 회의 관리
================================================================================

5.1 회의 생성
  - 워크스페이스 대시보드에서 "회의 관리" 섹션 확인
  - "새 회의 만들기" 버튼 클릭
  - 회의 정보 입력:
    * 회의 제목 (필수)
    * 회의 설명
    * 시작 시간 (날짜 및 시간 선택)
    * 종료 시간 (날짜 및 시간 선택)
    * 참여자 선택 (워크스페이스 멤버 중 선택)
  - "회의 생성" 버튼 클릭

5.2 회의 목록 보기
  - "목록 보기" 탭 선택
  - 회의 정보 표시:
    * 회의 제목
    * 시작/종료 시간
    * 참여자 수
    * 회의 설명
  - 회의 항목 클릭하여 상세 정보 확인

5.3 회의 상세 정보
  - "회의 정보" 탭에서 확인 가능한 정보:
    * 회의 제목
    * 회의 설명
    * 시작/종료 시간
    * 참여자 목록 (이메일, 역할)
    * 생성 날짜
    * 최종 수정 날짜

5.4 회의 수정
  - 권한: 회의 생성자만 가능
  - 회의 상세 페이지에서 "수정" 버튼 클릭
  - 회의 정보 변경
  - "저장" 버튼 클릭

5.5 회의 삭제
  - 권한: 회의 생성자만 가능
  - 회의 상세 페이지에서 "삭제" 버튼 클릭
  - 확인 다이얼로그에서 "삭제" 선택

5.6 회의 채팅 공유
  - 회의 상세 페이지에서 "채팅에 공유" 버튼 클릭
  - 공유할 채팅방 선택
  - "공유하기" 버튼 클릭
  - 채팅방에 회의 정보 카드 자동 전송
  - 회의 카드에는 제목, 시간, 참여자 수 표시

6.1 프로필 보기

  - 표시 정보:

6.2 프로필 편집
  - 프로필 영역의 "편집" 버튼 클릭
  - 수정 가능한 정보:
    * 사용자 이름
    * 프로필 사진 (URL)
  - "저장" 버튼 클릭

9. 실시간 기능
================================================================================

9.1 실시간 채팅
  - Supabase Realtime을 통한 실시간 메시지 동기화
  - 연결 상태 표시:
    * 초록색: 실시간 연결됨
    * 주황색: 연결 시간초과
    * 빨간색: 연결 실패
    * 파란색: 재연결 시도 중
    * 회색: 연결 종료

9.2 실시간 알림
  - 새 메시지 도착 시 자동 스크롤
  - 읽지 않은 메시지 카운트 표시
  - 채팅방 목록 실시간 업데이트

================================================================================
10. 파일 관리
================================================================================

10.1 파일 업로드 제한
  - 파일당 최대 크기: 10MB
  - 동시 업로드 최대 파일 수: 5개
  - 지원 형식:
    * 이미지: image/*
    * 문서: PDF, DOC, DOCX, TXT, MD
    * 압축: ZIP, RAR
    * 동영상: MP4, MOV, AVI

10.2 AI 파일 분석
  - Claude 모델: 이미지, PDF 분석 가능
  - OpenAI 모델: 이미지 분석 가능
  - 파일 첨부 후 AI 모드로 메시지 전송
  - AI가 파일 내용 분석 후 응답

================================================================================
11. 권한 체계
================================================================================

11.1 회사 권한
  - Owner (소유자):
    * 회사 생성자
    * 모든 권한 보유
  - Member (멤버):
    * 회사 가입 사용자
    * 워크스페이스 생성 및 가입 가능

11.2 워크스페이스 권한
  - 생성자:
    * 워크스페이스 생성자
    * 채팅방 생성 가능
    * 회의 관리 가능
  - Admin (관리자):
    * 채팅방 생성 가능
    * 멤버 관리 가능
  - Member (멤버):
    * 채팅 참여 가능
    * 회의 참여 가능

11.3 회의 권한
  - 생성자:
    * 회의 수정 및 삭제 가능
    * 채팅에 공유 가능
  - 참여자:
    * 회의 정보 조회 가능

================================================================================
12. 알아두면 좋은 기능
================================================================================

12.1 자동 기본 채팅방
  - 워크스페이스 생성 시 "일반" 채팅방 자동 생성
  - 기본 채팅방은 삭제 불가
  - 워크스페이스 멤버 자동 추가

12.2 읽음 상태 관리
  - 채팅방 선택 시 자동으로 읽음 처리
  - 읽지 않은 메시지는 빨간 점으로 표시

12.3 검색 기능
  - 회사 검색: 실시간 자동 검색 (300ms 디바운싱)
  - 워크스페이스 검색: 이름으로 필터링

12.4 시간 표시
  - 오늘 메시지: "오후 2:30" 형식
  - 어제 메시지: "어제 오후 2:30" 형식
  - 그 외: "10월 5일 오후 2:30" 형식

12.5 링크 처리
  - AI 메시지의 모든 링크는 새 탭에서 열림
  - 링크는 파란색으로 강조 표시
  - 마우스 호버 시 색상 변경

13. 문제 해결
================================================================================

13.1 로그인 문제
  - Google 로그인만 지원
  - 팝업 차단 해제 필요
  - 쿠키 및 로컬 스토리지 허용 필요

13.2 채팅방 목록이 보이지 않을 때
  - 워크스페이스에 가입했는지 확인
  - 페이지 새로고침 시도
  - 브라우저 콘솔에서 에러 확인

13.3 파일 업로드 실패
  - 파일 크기 확인 (10MB 이하)
  - 파일 형식 확인
  - 인터넷 연결 상태 확인

13.4 AI 응답이 오지 않을 때
  - AI 모드가 활성화되었는지 확인 (로봇 아이콘 파란색)
  - 모델 선택 확인
  - 잠시 후 다시 시도

13.5 실시간 연결 문제
  - 연결 상태 확인 (채팅방 헤더의 점 색상)
  - 페이지 새로고침
  - 인터넷 연결 상태 확인

16. 주의사항

16.1 데이터 보안

  - 파일은 Supabase Storage에 안전하게 저장
  - HTTPS 암호화 통신

16.2 파일 저장
  - 업로드된 파일은 영구적으로 저장
  - 삭제 기능은 현재 지원하지 않음

16.3 AI 사용 시 주의사항
  - AI 응답은 참고용으로만 사용
  - 중요한 결정은 사람이 최종 판단
  - 개인정보 또는 민감한 정보 입력 자제
