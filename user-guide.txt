================================================================================
1. 시작하기
================================================================================

1.1 로그인
----------
① 웹 브라우저에서 AXTIV 접속
② "Google로 로그인" 버튼 클릭
③ Google 계정 선택 및 인증
④ 자동으로 메인 페이지로 이동

※ 참고: 처음 로그인하는 경우 자동으로 계정이 생성됩니다.

1.2 로그아웃
-----------
① 워크스페이스 화면 좌측 하단의 프로필 영역 클릭
② 메뉴 버튼 (≡) 클릭
③ "로그아웃" 선택


================================================================================
2. 회사 가입 및 생성
================================================================================

2.1 회사 검색 및 가입
-------------------
① 로그인 후 "회사 검색" 페이지 접속
② 검색창에 회사명 입력 (자동 검색)
③ 검색 결과에서 원하는 회사 찾기
④ "가입하기" 버튼 클릭
⑤ 확인 후 자동으로 워크스페이스 페이지로 이동

※ 이미 다른 회사에 소속된 경우 변경 확인 메시지가 나타납니다.

2.2 새 회사 생성
---------------
① "회사 검색" 페이지 하단의 "새 회사 생성하기" 버튼 클릭
② 회사명 입력 (필수)
③ 회사 설명 입력 (선택)
④ "회사 생성" 버튼 클릭
⑤ 생성된 회사의 오너(Owner) 권한 자동 부여

※ 회사를 생성하면 자동으로 해당 회사에 소속됩니다.


================================================================================
3. 워크스페이스 관리
================================================================================

3.1 워크스페이스 가입
-------------------
① 회사 페이지에서 워크스페이스 목록 확인
② 상단 탭에서 "워크스페이스 검색" 선택
③ 검색창에 워크스페이스명 입력
④ 원하는 워크스페이스의 "가입하기" 버튼 클릭
⑤ "내 워크스페이스" 탭에서 가입 확인

3.2 워크스페이스 생성
-------------------
① 워크스페이스 페이지 상단의 "새로 만들기" 버튼 클릭
② 워크스페이스명 입력 (필수)
③ 워크스페이스 설명 입력 (선택)
④ "워크스페이스 생성" 버튼 클릭
⑤ 자동으로 "일반" 기본 채팅방 생성
⑥ 생성자는 자동으로 관리자(Admin) 권한 부여

3.3 워크스페이스 입장
-------------------
① "내 워크스페이스" 탭에서 가입된 워크스페이스 확인
② 원하는 워크스페이스 카드의 "워크스페이스 입장 →" 클릭
③ 워크스페이스 상세 페이지로 이동

3.4 워크스페이스 정보 확인
------------------------
각 워크스페이스 카드에서 확인 가능한 정보:
- 워크스페이스명
- 소속 회사명
- 멤버 수
- 내 역할 (생성자/관리자/멤버)
- 가입 여부


================================================================================
4. 팀 채팅
================================================================================

4.1 채팅방 목록 확인
------------------
① 워크스페이스 입장 후 좌측 사이드바 확인
② "팀 채팅" 섹션 펼치기 (클릭)
③ 가입한 채팅방 목록 표시

4.2 채팅방 선택 및 대화
--------------------
① 좌측 사이드바에서 원하는 채팅방 클릭
② 우측 채팅 화면에서 이전 메시지 확인
③ 하단 입력창에 메시지 작성
④ "전송" 버튼 클릭 또는 Enter 키 입력
⑤ 실시간으로 메시지 전송 및 수신

※ Shift + Enter: 줄바꿈

4.3 파일 업로드
-------------
① 메시지 입력창 우측의 📎 버튼 클릭
② 파일 선택 (여러 개 선택 가능)
③ 업로드된 파일 미리보기 확인
④ 메시지와 함께 또는 파일만 전송 가능
⑤ 전송 전 ❌ 버튼으로 파일 제거 가능

지원 파일 형식:
- 이미지: JPG, PNG, GIF, WEBP 등
- 문서: PDF, DOC, DOCX, TXT, MD
- 압축: ZIP, RAR
- 동영상: MP4, MOV, AVI

4.4 채팅방 생성
-------------
① "팀 채팅" 섹션의 "+" 버튼 클릭
② 채팅방 이름 입력
③ 채팅방 설명 입력 (선택)
④ "채팅방 생성" 버튼 클릭
⑤ 생성자는 자동으로 관리자 권한 부여

4.5 채팅방 나가기
---------------
① 채팅방 헤더의 "나가기" 버튼 클릭
② 확인 대화상자에서 "나가기" 클릭
③ 자동으로 기본 채팅방("일반")으로 이동

※ 기본 채팅방은 나갈 수 없습니다.

4.6 이전 메시지 불러오기
----------------------
① 채팅 화면 최상단으로 스크롤
② "이전 메시지 불러오기" 버튼 클릭
③ 과거 메시지 20개씩 추가 로드

5. 개인 메시지 (DM)
================================================================================

5.1 개인 메시지 시작
------------------
① 좌측 사이드바의 "개인 채팅" 섹션 펼치기
② 워크스페이스 멤버 목록 확인
③ 대화하고 싶은 사용자 클릭
④ 자동으로 1:1 채팅방 생성 또는 기존 대화 열기

5.2 개인 메시지 사용
------------------
- 팀 채팅과 동일한 방식으로 메시지 전송
- 파일 업로드 가능
- AI 채팅봇 사용 가능
- 실시간 메시지 동기화


================================================================================
6. AI 채팅봇 사용하기
================================================================================

6.1 AI 모드 활성화
----------------
① 메시지 입력창 우측의 🤖 버튼 클릭
② 버튼 색상이 보라색으로 변경되면 AI 모드 활성화
③ AI 모델 선택 버튼 표시 (기본: GPT-5)

6.2 AI 모델 선택
--------------
사용 가능한 AI 모델:
- GPT-5 (기본)
- GPT-4
- Claude 3.7 Sonnet
- Claude 3.5 Haiku
- Claude 3 Opus

① AI 모드 활성화 상태에서 모델 선택 드롭다운 클릭
② 원하는 AI 모델 선택

6.3 AI에게 질문하기
-----------------
① AI 모드 활성화
② 메시지 입력창에 질문 입력
③ "전송" 버튼 클릭
④ 사용자 메시지가 먼저 채팅에 표시됨
⑤ AI 응답이 스트리밍 방식으로 실시간 표시 (Claude)
   또는 전체 응답 표시 (GPT)

6.4 파일 분석 요청
----------------
① AI 모드 활성화
② 📎 버튼으로 분석할 파일 업로드
③ 질문 또는 지시사항 입력 (선택)
④ "전송" 버튼 클릭
⑤ AI가 파일을 분석하여 답변 제공

지원 파일 (AI 분석 가능):
- 이미지: JPG, PNG, GIF, WEBP
- PDF 문서 (Claude만 지원)

6.5 AI 모드 비활성화
------------------
① 🤖 버튼 다시 클릭
② 일반 채팅 모드로 전환


================================================================================
7. 미팅 관리
================================================================================

7.1 캘린더 확인
-------------
① 좌측 사이드바의 "미팅 관리" 섹션 펼치기
② 캘린더에서 회의가 있는 날짜 확인
   - 회의가 있는 날짜는 파란색 테두리 표시
③ 날짜 클릭 시 해당 날짜의 회의 목록 표시

7.2 회의 생성
-----------
① "미팅 관리" 섹션의 "회의 생성" 버튼 클릭
   또는 캘린더에서 날짜 선택 후 회의 관리 화면에서 생성
② 회의 제목 입력
③ 회의 설명 입력 (선택)
④ 시작 시간 설정
⑤ 종료 시간 설정
⑥ 참가자 선택 (워크스페이스 멤버)
⑦ "회의 생성" 버튼 클릭

7.3 회의 정보 확인
----------------
① 회의 관리 화면에서 "목록 보기" 탭 선택
② 회의 목록에서 확인하고 싶은 회의 클릭
③ "회의 정보" 탭으로 자동 전환
④ 회의 상세 정보 확인:
   - 제목, 설명
   - 시작/종료 시간
   - 참가자 목록 및 역할
   - 생성 날짜

7.4 회의 수정
-----------
① "회의 정보" 탭에서 "수정" 버튼 클릭
② 회의 정보 변경
③ 참가자 추가/제거
④ "저장" 버튼 클릭

※ 회의 생성자와 관리자만 수정 가능합니다.

7.5 회의 삭제
-----------
① "회의 정보" 탭에서 "삭제" 버튼 클릭
② 확인 대화상자에서 "삭제" 클릭

※ 회의 생성자와 관리자만 삭제 가능합니다.

7.6 회의 공유
-----------
① "회의 정보" 탭에서 "공유" 버튼 클릭
② 공유 링크 또는 초대 메시지 확인
③ 복사하여 다른 사용자와 공유

================================================================================

8.1 대시보드 접근
---------------

9. 프로필 설정
================================================================================

9.1 프로필 확인
-------------
① 좌측 사이드바 하단의 프로필 영역 클릭
② 메뉴 버튼 (≡) 클릭
③ "프로필 보기" 선택

9.2 프로필 정보 확인
------------------
프로필에서 확인 가능한 정보:
- 프로필 이미지
- 이름
- 직책 및 부서
- 워크스페이스 역할 (관리자/멤버)
- 회사 역할 (오너/관리자/멤버)
- 업무 상태 (업무 가능/바쁨/자리비움/회의중)
- 이메일
- 전화번호
- 자기소개
- 워크스페이스 가입일

9.3 프로필 수정
-------------
① 프로필 다이얼로그 우측 상단의 ✏️ 버튼 클릭
② 수정 가능한 항목:
   - 프로필 이미지 (클릭하여 업로드)
   - 전화번호
   - 부서
   - 직책
   - 자기소개
   - 업무 상태
③ 수정 완료 후 ✓ 버튼 클릭
④ 저장 확인 메시지 표시

9.4 프로필 이미지 변경
--------------------
① 프로필 수정 모드에서 프로필 이미지 클릭
② 이미지 파일 선택 (JPG, PNG, WEBP)
③ 미리보기 확인
④ 저장하여 업로드

※ 이미지 크기는 5MB 이하여야 합니다.

9.5 업무 상태 설정
----------------
사용 가능한 상태:
- 🟢 업무 가능 (기본)
- 🔴 바쁨
- 🟡 자리비움
- 📅 회의중

① 프로필 수정 모드에서 상태 드롭다운 선택
② 원하는 상태 클릭
③ 저장

================================================================================
문제 해결
================================================================================

Q: 로그인이 안 돼요
A: Google 계정 인증을 완료했는지 확인하세요. 팝업 차단이 되어있지 않은지
   브라우저 설정을 확인하세요.

Q: 회사/워크스페이스가 보이지 않아요
A: 페이지를 새로고침하거나, 로그아웃 후 다시 로그인해보세요.

Q: 파일 업로드가 안 돼요
A: 파일 형식과 크기를 확인하세요. 네트워크 연결 상태를 확인하세요.

Q: AI가 응답하지 않아요
A: AI 모드가 활성화되어 있는지 확인하세요 (🤖 버튼이 보라색).
   네트워크 연결 상태를 확인하세요.

Q: 채팅방을 나가고 싶어요
A: 채팅방 헤더의 "나가기" 버튼을 클릭하세요.
   기본 채팅방("일반")은 나갈 수 없습니다.

Q: 회의를 수정/삭제할 수 없어요
A: 회의 생성자 또는 관리자 권한이 필요합니다.
